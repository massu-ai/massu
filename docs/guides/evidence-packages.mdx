# Evidence Packages

Generate comprehensive compliance evidence packages as downloadable PDFs for your audit needs.

## What's Included

Each evidence package contains:

- **Cover page** with organization name, report type, and reporting period
- **Executive summary** of compliance status
- **Audit chain certificate** proving audit trail integrity
- **Change log evidence** with all changes and approvals
- **Policy compliance summary** with violation details and resolutions

## Generating a Package

1. Navigate to **Compliance > Evidence** in your dashboard
2. Select the reporting period (start and end dates)
3. Choose the report type (SOC2, ISO 27001, or Custom)
4. Click **Generate Package**
5. Download the PDF when generation is complete

## Supported Formats

- **SOC2** — Structured evidence for SOC2 Type II audits
- **ISO 27001** — Evidence aligned with ISO 27001 controls
- **Custom** — Configurable sections for other compliance frameworks
