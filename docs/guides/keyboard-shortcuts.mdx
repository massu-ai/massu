# Keyboard Shortcuts

The Massu AI Cloud dashboard supports keyboard shortcuts for faster navigation. Press `?` at any time to see available shortcuts.

## Navigation Shortcuts

All navigation shortcuts use a chord pattern: press `g` followed by a letter.

| Shortcut | Action |
|----------|--------|
| `g` then `d` | Go to Dashboard overview |
| `g` then `s` | Go to Sessions |
| `g` then `t` | Go to Team |
| `g` then `c` | Go to Cost Intelligence |
| `g` then `b` | Go to Billing |

## Action Shortcuts

| Shortcut | Action |
|----------|--------|
| `/` | Focus the search bar |
| `?` | Show keyboard shortcuts help |
| `Escape` | Close modal or dismiss |

## Notes

- Keyboard shortcuts are disabled when you're typing in an input field, textarea, or content-editable element
- Chord shortcuts (like `g` then `d`) have a 1-second timeout â€” if you don't press the second key within 1 second, the chord is cancelled
- Keyboard shortcuts are available on Cloud Pro plans and above
