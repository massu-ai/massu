# Scheduled Reports

Automate compliance report generation on a recurring schedule with email distribution to stakeholders.

## Configuring a Schedule

1. Navigate to **Compliance > Schedules** in your dashboard
2. Click **Create Schedule**
3. Configure:
   - **Report type**: SOC2, ISO 27001, or Custom
   - **Schedule**: Weekly, Monthly, or Quarterly
   - **Recipients**: Email addresses for report distribution
   - **Sections**: Choose which sections to include

## Report Templates

### SOC2
Includes: Executive summary, access controls, change management, incident response, data protection.

### ISO 27001
Includes: Information security policy, risk assessment, access control, cryptography, physical security.

### Custom
Select from available sections to build a report tailored to your compliance needs.

## Distribution

When a report is generated, all configured recipients receive an email with:
- Report summary
- Key metrics (compliance score, violations, changes)
- Download link for the full report
